# Data Analysis Portfolio

This portfolio showcases applied data analysis projects focused on pricing dynamics in transportation markets. 
It combines web scraping, data cleaning, exploratory analysis, and econometric modeling to extract insights from real-world data.

---

## 📊 FlixBus Pricing Analysis

This project analyzes dynamic pricing strategies using scraped FlixBus data, supported by a structured econometric framework developed in an 
accompanying document. The analysis follows the methodology of *"Dynamic Pricing and Seat Availability in Intercity Transport Markets"*.
Results show non-linear price adjustments over time and heterogeneity across routes. Findings were validated against equivalent estimations in R, 
showing consistent patterns and robust interpretation.

---

## ✈️ Ryanair Web Scraping & Pricing Analysis

This project involves building a custom web scraper using Selenium to collect real-time flight data from Ryanair. 
The dataset was cleaned, standardized (including multi-currency price conversion), and enriched with key variables 
such as booking horizon and flight duration. The analysis explores pricing patterns across routes and time, highlighting 
evidence of dynamic pricing behavior.
